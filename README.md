# Motor Insurance Project

## Project Overview
This project analyses a Vehicle insurance company data to identify the most profitable brands, the most profitable Vehicle type, the vehicles taking highest number of policies but worse Premium/Claim ratio etc.
An interactive dashboard is created using Power BI to help stakeholders make data-driven decisions.

## Objectives
- Analyze company's performance over the years
- Identify top performing vehicle brands and vehicle type
- Measure Premium vs Claim ratio

## Tools & Technologies Used
- Power BI
- Microsoft Excel

## Dataset Information
The dataset includes the following columns:
- Insurance begin date
- Insurance end date
- Insured value
- Premium
- Vehicle type
- Vehicle Brand
- Vehicle usage
- Claim paid

## Key Insights
* Motor Cycle owners take the highest number of policies
*	Highest number of claims are claimed by vehicles used for General Cartage
*	2018 is the most profitable year having highest Premium/Claim ratio of 114%
*	2014 and 2015 were the worst performing years having Premium/Claim ratio of only 34%
*	Toyota brand vehicles buy highest number of policies.
*	Fire fighting vehicles has highest Premium/Claim ratio overall
*	Massy Fureguson has been the profitable brand for the business over the years
---

## Dashboard Features
- Cards for Important business KPIs
- Ribbon chart for Vehicle Usage vs Claims
- Bar Chart for Vehicle type vs Policies opened
- Scatter chart for Avg Premium vs Avg claim
- Interactive slicers
